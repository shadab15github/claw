# claw-site MCP - Test Prompts

Use these prompts after adding `claw-site` to Claude Desktop.

## `extract_urls`

```
Using claw-site, extract URLs from https://example.com.
```

```
Using claw-site, extract URLs from https://quotes.toscrape.com with same_domain=true, same_path=true, and limit=25.
```

```
Using claw-site, extract URLs from https://www.python.org with same_domain=false, same_path=false, and limit=100.
```

## `extract_content`

```
Using claw-site, extract the page content from https://example.com.
```

```
Using claw-site, extract the page content from example.com without including the title.
```

```
Using claw-site, extract the readable Markdown content from https://quotes.toscrape.com.
```

## `extract_image_text`

```
Using claw-site, extract text from the image at C:\path\to\image.png.
```

```
Using claw-site, extract text from the image URL https://example.com/image.png.
```

```
Using claw-site, extract text from a base64-encoded image using lang=eng.
```
